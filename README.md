# ZeroTierMenu

Небольшое приложение для macOS, которое живёт в строке меню и помогает работать с вашей сетью ZeroTier.

## Что умеет

- показывает значок в строке меню macOS
- автоматически определяет все активные ZeroTier-сети
- если активна одна сеть, показывает её сразу
- если активных сетей несколько, показывает их все и даёт выбрать сеть в выпадающем списке для перехода в ZeroTier Central
- определяет локальные ZeroTier-подсети
- сканирует живые хосты во всех найденных подсетях
- показывает имя хоста и IPv4-адрес
- копирует IPv4-адрес в буфер обмена по клику
- позволяет задавать имена хостам вручную
- позволяет добавлять хосты вручную
- показывает статус хоста через цветную точку
- показывает, к какой сети относится хост, если сетей несколько
- умеет открывать страницу сети в ZeroTier Central по клику на `network id`

## Запуск

```bash
./script/build_and_run.sh
```

## Установка

После сборки приложение можно установить так:

```bash
./script/install_to_applications.sh
```

Скрипт:

- копирует приложение в `/Applications`
- удаляет `.app` из `dist`, чтобы не было дубля

`/Applications/ZeroTierMenu.app`

Пользовательский конфиг хранится отдельно, поэтому при обновлении приложения не теряется.

## Как пользоваться

1. Запустите приложение.
2. Нажмите на значок в строке меню.
3. Если активных сетей несколько, при необходимости выберите нужную в выпадающем списке в шапке.
4. Нажмите `Сканировать сеть`, чтобы найти живые хосты во всех активных ZeroTier-сетях.
5. Кликните по IPv4-адресу, чтобы скопировать его.
6. При необходимости задайте хосту имя вручную или добавьте новый хост снизу списка.

## Ограничения

- Имена хостов автоматически определяются только если в сети настроен reverse DNS.
- Автозапуск использует системный механизм macOS и может требовать подтверждение системы.

## Конфиг

Пользовательские имена хостов и вручную добавленные хосты сохраняются в:

`~/Library/Application Support/ZeroTierMenu/config.json`
