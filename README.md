# ZeroTierMenu

Небольшое приложение для macOS, которое живёт в строке меню и помогает работать с вашей сетью ZeroTier.

## Что умеет

- показывает значок в строке меню macOS
- работает с сетью `a581878f7d73f059`
- определяет локальную ZeroTier-подсеть
- сканирует живые хосты в этой подсети
- показывает имя хоста и IPv4-адрес
- копирует IPv4-адрес в буфер обмена по клику
- позволяет задавать имена хостам вручную
- позволяет добавлять хосты вручную
- показывает статус хоста через цветную точку
- умеет открывать страницу сети в ZeroTier Central по клику на `network id`

## Запуск

```bash
./script/build_and_run.sh
```

## Как пользоваться

1. Запустите приложение.
2. Нажмите на значок в строке меню.
3. Нажмите `Сканировать сеть`, чтобы найти живые хосты.
4. Кликните по IPv4-адресу, чтобы скопировать его.
5. При необходимости задайте хосту имя вручную или добавьте новый хост снизу списка.

## Ограничения

- Имена хостов автоматически определяются только если в сети настроен reverse DNS.
- Автозапуск использует системный механизм macOS и может требовать подтверждение системы.
